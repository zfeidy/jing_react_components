## 利用wnpm发布组件 ##
```shell
npm install -g wnpm --registry=http://npm.caimi-inc.com
```

## 安装组件 ##
```shell
npm install @wac/wacai_react_components
```

## 组件使用 ##
* commonJS方式
```javascript
import { WcTable } form '@wac/wacai_react_components';
```

* 前端调用
```javascript
<script src="src/dist/index.js"></script>
```
